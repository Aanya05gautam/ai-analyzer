# AI Analyzer

AI Analyzer is a **multi-page React web application** built with **JSX and TailwindCSS**.  
It allows users to log in, analyze text, and learn about the app in an interactive way.

## Features

- Multi-page app: Home, Analyzer, About
- Login form with interactive messages
- Text Analyzer with word & character count
- Responsive design with TailwindCSS
- Gradient navbar and aesthetic colors

## Tech Stack

- React (JSX)
- React Router DOM
- TailwindCSS
- Vite (for development)
